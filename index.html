Connect to Strava

Strava will auto-detect completed workouts and show your activity stats.
You'll need a free Strava API app — takes 2 minutes to create.

Setup steps:
1. Create an app at strava.com/settings/api
2. Set "Authorization Callback Domain" to localhost
3. Serve this folder: python3 -m http.server 8080
4. Open http://localhost:8080
